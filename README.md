## **개발 개요**
버스정류장에서 버스를 기다리는데, 버스가 그냥 가버리는 경우가 있지요?\
스마트폰을 하다가 내려야 할 정류장을 지나가버리는 경우가 있지요?\
안개가 많이 낀 날, 버스가 오는데 번호가 잘 보이지 않지요?\
이러한 문제들을 해결하고자 **StopBus**는 버스를 제.대.로. 세우고자 합니다.
